광고주 카테고리

광고주 그룹

여러 광고주명을 하나의 그룹으로 묶어서 갤러리/필터/분석에서 그룹명으로 볼 수 있습니다. (예: "삼성화재 다이렉트" + "삼성화재 생명보호" → "삼성화재")

자사 설정

여기서 선택한 광고주(또는 그룹)만 "자사"로 인식되어 갤러리/필터/분석의 자사·경쟁사 구분에 반영됩니다. 선택하지 않으면 기존 수집 결과의 구분을 그대로 사용합니다.

AI 분류 설정

광고 소재 자동 분류에 사용할 AI를 선택하고 API 키를 입력하세요. 저장 후 다음 분류 실행부터 적용됩니다. 양쪽 키를 모두 입력해두면 프로바이더만 바꿔서 전환할 수 있습니다.
